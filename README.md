# EmiOS-Web

The marketing landing page for **EmiOS**, served at [emios.ai](https://emios.ai) via Cloudflare Pages.

## Files
- `index.html` — the live landing page (emoji feature grid).
- `index_v2.html` — an alternate screenshot-based redesign (not currently the live page).
- `style.css`, `main.js` — shared styles and slideshow script.
- `*.png` / `*.jpg` — UI screenshots used by `index_v2.html`.

## Deploy
Cloudflare Pages serves this repo's root directly (static files, no build step). Pushing to `main` deploys to emios.ai.

GitHub links on the page point to the public app repo, [EmiOS-Beta](https://github.com/SemiSimpleMath/EmiOS-Beta).
